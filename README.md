# API

## Method
    POST https://censor.yablach.co/

## Example Request
  ```json
  {
    "..." : "...",
    "image": "url",
    "headers" : {},
    "cookies" : {}
  }
  ```

## Example Response
  ```json
  {
    "..." : "...",
    "image": "url",
    "headers" : {},
    "cookies" : {},
    "censor": true
  }
  ```
