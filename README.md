# API

## Method
    POST https://censor.yablach.co/

## Example Request
  ```json
  {
    "..." : "...",
    "image": "https://2ch.hk/X/src/XXX/XXX.jpg|png",
    "..." : "..."
  }
  ```

## Example Response
  ```json
  {
    "..." : "...",
    "image": "https://2ch.hk/X/src/XXX/XXX.jpg|png",
    "censor": true | false,
    "..." : "..."
  }
  ```
